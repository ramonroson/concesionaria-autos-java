# Sistema de Gestión de Concesionaria

Aplicación de escritorio desarrollada en Java para la administración de automóviles en una concesionaria.

## Descripción

Este proyecto fue desarrollado como práctica para afianzar conocimientos de Programación Orientada a Objetos, interfaces gráficas con Java Swing y persistencia de datos mediante JPA.

La aplicación permite registrar vehículos, consultar la información almacenada, modificar registros existentes y eliminar automóviles de la base de datos a través de una interfaz gráfica sencilla e intuitiva.

## Funcionalidades

* Alta de automóviles.
* Consulta de vehículos registrados.
* Edición de información existente.
* Eliminación de registros.
* Persistencia de datos utilizando JPA.

## Tecnologías utilizadas

* Java
* Java Swing
* JPA (Java Persistence API)
* Maven
* MySQL
* Git y GitHub

## Arquitectura del proyecto

El proyecto se encuentra organizado por capas para separar responsabilidades:

* **igu**: contiene las ventanas y componentes gráficos.
* **logica**: contiene las reglas de negocio de la aplicación.
* **persistencia**: administra el acceso a datos mediante JPA.
